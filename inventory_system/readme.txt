Система инвентаря.

Компоненты:
  ItemComponent - компонент предмета.
    stackable - (bool) - стакается ли предмет.
    stack_size - (int) - кол-во предметов в стаке. По умолчанию 64.

  InventoryComponent - компонент инвентаря
    max_volume - (int) - максимальное кол-во предметов в инвентаре.
    items - (list) - список предметов в инвентаре вида (item_id, count).
    free_volume - (int) - свободное место в инвентаре.

    item_in_inventory(item_id, count=1) - (func) -
          возвращает True, если в инвентаре
          присутствует заданный предмет в заданном кол-ве.

Системы:
  InventorySystem - система инвентаря.
    Принимаемые события:
      AddItemToInventoryEvent - добавляет предмет в инвентарь
        inventory_id - айди сущности, которой нужно добавить предмет
        item_id - айди предмета, который нужно добавить
        count - кол-во предметов.

      RemoveItemFromInventoryEvent - удаляет предмет из инвентаря
        inventory_id - айди сущности, которой нужно добавить предмет
        item_id - айди предмета, который нужно добавить
        count - кол-во предметов.

    Генерируемые события:
      EntityDoesNotExistEvent - сущности не существует
        entity_id - айди сущности

      EntityHasNo%sEvent - сущность не имеет компонента, соответствующего системе (InventoryComponent)
        entity_id - айди сущности

      EntityHasNoItemComponentEvent - сущность не имеет компонента ItemComponent
        entity_id - айди сущности

      AddItemSucsessEvent - предмет успешно добавлен в инвентарь.
        inventory_id - айди инвентаря (сущности), в который был добавлен предмет
        item_id - айди предмета, который был добавлен в инвентарь
        count - кол-во предмета, который был добавлен

      InventoryHasNoFreeSpaceEvent - в инвентаре нет свободного места
        inventory_id - айди инвентаря (сущности), в который не получилось добавить предмет
        item_id - айди предмета, который не получилось добавить
        count - кол-во предмета

      RemoveItemSucsessEvent - предмет был успешно удален из инвентаря
        inventory_id - айди инвентаря (сущности), из которого был удален предмет
        item_id - айди предмета, который был удален
        count - кол-во предмета

      InventoryHasNoItemsEvent - в инвентаре недостаточно предметов для удаления
        inventory_id - айди инвентаря (сущности), из которого не получилось удалить предмет
        item_id - айди предмета, который не получилось удалить
        count - кол-во предмета
